CSCraft Playground ready. Press Ctrl+Enter to run.

CSCraft transpiles C# → Java (Fabric API) so you can write Minecraft mods in C#.
Runs entirely in your browser via WebAssembly. No data is sent to any server.

The Download button gives you a ready-to-build Gradle project.
Run ./gradlew build inside it to get the .jar.
