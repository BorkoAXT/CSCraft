Quick Start

CSCraft automates everything. Run these commands to begin:

dotnet new classlib -n MyMod
cd MyMod
dotnet add package CSCraft.Sdk
dotnet build

The build engine automatically discovers your **JDK 21** and generates the Fabric environment.

Player API

MethodDescription
SendMessage(string)Sends chat message
Teleport(x, y, z)Moves player instantly
GiveItem(id, count)Adds items to inventory
Heal(float)Restores health points

Command Registration

Registering commands is handled via McCommand. Arguments are parsed automatically.

McCommand.RegisterWithPlayer("greet", "target", (src, target) => {
    src.SendMessage($"You greeted {target.Name}!");
    target.SendMessage($"{src.Player.Name} says hi!");
});

Troubleshooting

JDK Not Found: Version 1.1.0+ uses Java Discovery. If it fails, manually set path in .csproj:

<CSCraftJavaHome>C:\Java\jdk-21</CSCraftJavaHome>
